Έναρξη Σεμιναρίου Mobile και Web Εφαρμογών

Κατά τη διάρκεια του σεμιναρίου οι ενδιαφερόμενοι φοιτητές θα έχουν τη δυνατότητα να παρακολουθήσουν σειρά τουλάχιστον 5 διαλέξεων γύρω από τις σύγχρονες τεχνολογίες αιχμής στον τομέα του Δικτυακού Προγραμματισμού.

Η ύλη θα περιλαμβάνει διάφορα εργαλεία, γλώσσες και τεχνολογίες. Ενδεικτικά αναφέρονται: LAMP, noSQL, HTML5, CSS3, AJAX, Javascript, Node.JS, REST APIs, CURL και noJava Android development.

Εισηγητής του σεμιναρίου θα είναι ο Απόστολος Αναγνωστόπουλος. Σε επόμενες συναντήσεις ενδέχεται να γίνουν παρουσιάσεις και από άλλους εισηγητές και πιθανόν φοιτητές.

Επειδή το περιεχόμενο των διαλέξεων θα ακολουθεί κάποια λογική σειρά είναι καλό η παρακολούθηση να είναι τακτική ώστε να μην δημιουργούνται κενά στους φοιτητές που παρακολουθούν.

Το σεμινάριο θα ολοκληρωθεί με την ανάπτυξη project και από τους συμμετέχοντες.
